## Methods

### :Name()

<ParamField path="returns" type="string">
  Returns the name of the group.
</ParamField>

### :Parent()

<ParamField path="returns" type="CThirdTab">
  Returns the parent of the group.
</ParamField>

### :Type()

<ParamField path="returns" type="Enum.WidgetType">
  Returns the widget type of the group.
</ParamField>

### :Open()

<ParamField path="returns" type="nil">
  Opens the parent tabs and expands the group.
</ParamField>

### :Slider(sliderName: `string`, minValue: `integer`, maxValue: `integer`, defaultValue: `integer`, format [?]: `string|function`): `CMenuSliderInt`

Creates a new integer slider widget.

<ParamField path="sliderName" type="string" required="true">
  The name of the slider.
</ParamField>

<ParamField path="minValue" type="integer" required="true">
  Minimum value of the slider.
</ParamField>

<ParamField path="maxValue" type="integer" required="true">
  Maximum value of the slider.
</ParamField>

<ParamField path="defaultValue" type="integer" required="true">
  Default value of the slider.
</ParamField>

<ParamField path="format" type="string|function" default="%d">
  Format string or function for the slider value.
</ParamField>

**Example:**
```lua
group:Slider("slider", 0, 100, 50, "%d")  -- Creates an integer slider
```

